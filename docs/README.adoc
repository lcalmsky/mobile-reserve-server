.Overview
****
쿠폰 발급 및 조회를 위해 아래 API를 사용합니다.

----
1. 쿠폰 발급 API: 휴대폰 번호 별로 고유한 쿠폰 번호를 발급합니다. // <1>

2. 쿠폰 조회 API: 발급된 쿠폰을 조회합니다. // <1>
----
<1> ``쿠폰 발급 API``, ``쿠폰 조회 API``는 API 방식으로 쿠폰 서버와 연동합니다.
****
